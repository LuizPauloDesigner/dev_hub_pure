# 🚀 Pure Dev Dashboard - O Seu Espaço Criativo Definitivo

## 🏗️ Sobre o Projeto
Uma Central de Comando de alta performance e produtividade para desenvolvedores e criativos. Este dashboard integra ferramentas de planejamento visual, gestão de tarefas e sincronização cloud.

## ✨ Destaque: Mural Criativo (Milanote Clone)
O módulo **Mural Criativo** foi reconstruído para oferecer uma experiência de planejamento visual de elite:
- **Infinity Canvas**: Zoom e Pan suaves para projetos complexos.
- **Conexões Inteligentes**: Linhas que se adaptam e seguem os blocos automaticamente.
- **Aninhamento Real**: Arraste elementos para dentro de colunas para organização instantânea.
- **Smart Guides**: Alinhamento milimétrico automático com guias visuais.
- **Minimap & Navigation**: Localize-se rapidamente em grandes espaços de trabalho.

## 🛠️ Tecnologias
- **Frontend**: Vite, React 18, TypeScript, Tailwind CSS, shadcn/ui.
- **Backend/DB**: Node.js, Express, SQlite (com persistência via IndexedDB local).
- **Design System**: Design modular com suporte nativo a Dark Mode e temas dinâmicos.

## 🚀 Como Rodar Localmente

1. **Instale as dependências**
   ```sh
   npm install
   ```
2. **Execute o servidor de desenvolvimento**
   ```sh
   npm run dev
   ```

## 🌐 Deployment
Este projeto está preparado para ser hospedado em qualquer provedor estático (Frontend) e Node.js (Backend).
