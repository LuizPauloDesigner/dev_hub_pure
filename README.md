# Pure Dev Dashboard

## Project Info
Central de Comando e produtividade para desenvolvedores B2C e B2B SaaS.

## How to Run Locally

1. **Clone the repository**
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Run the development server**
   ```sh
   npm run dev
   ```

## Technologies
- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn-ui
- **Backend**: Node.js, Express, Better-SQLite3
- **Database**: SQLite (Sync Cloud)

## Deployment
This project can be deployed to any static hosting provider (frontend) and Node.js compatible server (backend).
